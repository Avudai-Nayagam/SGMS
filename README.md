SGMS is a robust Java-based management portal designed to streamline the organization of sports events. It provides dedicated interfaces for administrators, players, and team managers to interact with game schedules, results, and announcements.

## 🚀 Features

### 🛠 Admin Portal
* **Game Management:** Create and finalize both individual and team-based games.
* **Announcements:** Send official updates to all users via the Admin Announce module.
* **System Oversight:** Manage the overall database of players, teams, and game schedules.

### 👤 Player Portal
* **Registration:** Sign up for specific individual or team games.
* **Personal Dashboard:** View personal game schedules and results.
* **Communication:** Access player-specific messages and announcements.

### 👥 Team Management
* **Roster Control:** Register teams and manage player participation.
* **Performance Tracking:** View team-specific boards, schedules, and results.

## 📂 Project Structure

The project follows a modular Java Swing/AWT structure:
* **Forms (.form):** UI layout files for the graphical interface.
* **Source (.java):** Backend logic for game analysis, user sessions, and database interaction.
* **Modules:**
    * `Login/Sign-up`: Secure access for different user roles.
    * `View_LeaderBoard`: Real-time ranking updates.
    * `Schedule`: Dynamic viewing of upcoming matches.

## 🛠 Installation & Setup

1. **Prerequisites:** 
   * Java Development Kit (JDK) 8 or higher.
   * An IDE like NetBeans (recommended, as it uses the `.form` files) or IntelliJ IDEA.
2. **Clone the repository:**
   ```bash
   git clone https://github.com
   ```
3. **Run the Application:**
   * Open the project in your IDE.
   * Locate `Welcome.java` or `Login_Page.java`.
   * Right-click and select **Run File**.
